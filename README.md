# Project 2: VoIP Gateway

**Course:** ECEN-5803 Master Embedded Systems Architecture<br>
**Authors:** Hannah Caldwell-Meurer, Nimish Sabnis, and Nimisha Madapura Rajashekhar 

---

## Overview

This repository contains the joint project between the authors simulating a **VoIP Gateway** embedded product. The project is designed to showcase key concepts in embedded systems development, combining both hardware interaction and software architecture. Utilizing Raspberry Pi 4 Model Quad ARM Cortex A72 with QNX and Linux

The final project is divided into **7 modules**, each building on the previous to introduce new techniques, concepts, and tools relevant to embedded systems engineering.
